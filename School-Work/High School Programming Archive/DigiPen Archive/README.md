This folder stores all my old Godot projects. I was generally involved in the debugging and asset creations for all projects present here. 

The two pre-zipped folders are 2D games that are small enough for GitHub to accept in their entirety. However, the one 3D game wasn't very space-efficient so I broke it into 100MB clusters for GitHub not to throw a fit.
I recommend using WinRAR to decompress all of those files, just select them all and tell it to 'Extract Here' to bring it all back together.
